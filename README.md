# E-Shop | PHP & MySQL E-commerce Project

**E-Shop** is a beginner-friendly e-commerce web application built with **PHP, MySQL, HTML, CSS, and JavaScript**.  
It allows users to **register, login, browse products, and manage their profiles**. Admin users can **add, edit, or delete products** and view registered users.

---

## Features

- User registration and login with password hashing
- Profile update and password change
- Browse products with search functionality
- Admin panel with product and user management
- Dark/Light mode toggle using cookies
